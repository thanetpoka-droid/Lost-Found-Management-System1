# 🔍 Lost & Found Management System

A Java-based application for managing lost and found items. This project demonstrates Object-Oriented Programming (OOP) concepts and basic CRUD operations with a MySQL database.

## 🛠 Tech Stack
- Java
- MySQL
- JDBC
- VS Code

## ✨ Features
- Add lost and found items
- View all records
- Update item information
- Delete records
- Search items by ID

## 📂 Project Structure
- `src/` – Java source code
- `bin/` – Compiled class files
- `lib/` – MySQL JDBC driver
